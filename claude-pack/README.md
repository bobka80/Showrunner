# Claude project pack

Generated by **`node create-repomix.js`** — do not commit the large output files.

Also started **automatically in the background** after every successful **`node milestone.js`**. Skip with `node milestone.js "note" --no-repomix`. Log: `claude-pack/repomix-last-run.log`.

## What this is

A **Repomix** bundle of Showrunner source + `docs/ai/` for upload to a **Claude / quote.ai project knowledge** tab. Default output is **~2 MiB parts** so the mix fits project upload limits (a single ~5 MB file is often rejected).

## Commands

**Director trigger in Cursor:** say **create repo mix** — the agent runs the pack and lists every part path.

```bash
node create-repomix.js              # curated, ~2 MiB parts (recommended)
node create-repomix.js --split 1mb  # smaller parts if a UI still rejects
node create-repomix.js --no-split   # one monolith (~5 MB — usually too large)
node create-repomix.js --full         # includes vendor Javadoc (~11M tokens — usually too large)
```
