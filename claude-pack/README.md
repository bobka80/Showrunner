# Claude project pack

Generated by **`node create-repomix.js`** — do not commit the large output files.

## What this is

A **Repomix** bundle of Showrunner source + `docs/ai/` for upload to a **Claude project knowledge** tab. Lets you brainstorm on mobile with full code context (Firebase hosting, APK pipeline, GAS payload split, etc.) without burning chat context on every message.

## Commands

```bash
node create-repomix.js              # curated ~1M tokens (recommended)
node create-repomix.js --split 2mb  # multiple parts if upload size limit
node create-repomix.js --full         # includes vendor Javadoc (~11M tokens — usually too large)
```

## Upload

1. Run the command on your PC (repo must be current).
2. Upload `repomix-output.md` (or `repomix-output_001.md`, …) to Claude project knowledge.
3. Optionally upload `instructions.md` (navigation + current priorities).

## Cost

- **Packing:** free (local CPU, no API tokens).
- **Claude chat:** normal usage; project knowledge is retrieved per message, not full 1M tokens every turn.
