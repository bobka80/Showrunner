# Claude project pack

Generated by **`node create-repomix.js`** — do not commit the large output files.

Also started **automatically in the background** after every successful **`node milestone.js`** (curated single file). Skip with `node milestone.js "note" --no-repomix`. Log: `claude-pack/repomix-last-run.log`.

## What this is

A **Repomix** bundle of Showrunner source + `docs/ai/` for upload to a **Claude project knowledge** tab. Lets you brainstorm on mobile with full code context (Firebase hosting, APK pipeline, GAS payload split, etc.) without burning chat context on every message.

## Commands

**Director trigger in Cursor:** say **create repo mix** — the agent runs the pack and tells you the drag-drop path.

```bash
node create-repomix.js              # curated ~1M tokens (recommended; one file only)
node create-repomix.js --split 2mb  # multiple parts if upload size limit
node create-repomix.js --full         # includes vendor Javadoc (~11M tokens — usually too large)
```

## Drag & drop location

After the command finishes (or after a milestone), upload from:

**`S:\Gdrive\3Showrider\Code Env\claude-pack\repomix-output.md`**

*(path is under your repo root → `claude-pack/` folder)*

Optional: `claude-pack/instructions.md`
